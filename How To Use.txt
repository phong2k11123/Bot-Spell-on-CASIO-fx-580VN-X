# 📖 Hướng Dẫn Sử Dụng Tool **Spell** Trên CASIO fx-580VN X ⚙

## I. Cấu trúc dự án

```
📂 Bot - Spell on CASIO fx-580VN X
│
├── chars.txt          # Chứa danh sách ký tự gốc
├── takechars.txt      # Chứa dữ liệu ký tự đã lấy
└── Spell-CASIO.py     # File chạy chính của Tool
```

---

## II. Cách chạy file Spell

1. **Đặt file** `Spell-CASIO.py` **trong cùng thư mục** với `chars.txt` và `takechars.txt`.
2. **Chạy file** `Spell-CASIO.py` bằng Python.
3. Làm theo hướng dẫn hiển thị trên màn hình.

---

## III. Hướng dẫn sử dụng

* **Nhập `1`** → Đọc phần **lưu ý sử dụng**.
* **Nhập `2`** → **Liên hệ** với tác giả.
* **Nhập `3`** → Bắt đầu **spell ký tự**.

---
